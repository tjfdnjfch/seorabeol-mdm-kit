# 서라벌중학교 디지털 키트
이 저장소는 수업용 앱, 기기 사용 가이드, MDM 설정 파일 등
학교 내 디지털 튜터 업무를 위한 자료를 통합 관리하기 위해 만들어졌습니다.


## 포함 파일
- APK 파일 (디벗용 설치)
- 전자칠판/충전함/디벗 매뉴얼
- 안내문 및 배포용 문서
- 배포용 학습자료

## 구조 (예시)
```
seorabeol-mdm-kit/
│
├── README.md
├── /apk/
│   └── [앱명].apk          ← 수업용 앱
├── /manuals/
│   └── [기기 매뉴얼].pdf
├── /configs/
│   └── mdm_whitelist.txt  ← MDM 앱 화이트리스트
├── /notices/
│   └── 안내장, 가정통신문 등
```
